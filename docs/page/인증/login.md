# 로그인

로그인 페이지는 사용자가 서비스를 이용하기 위해  
가장 먼저 접근하는 인증 화면입니다.

프로젝트에서는 **학교 시스템과 연동된 일반 로그인**을 기본으로 제공하며,  
로그인 이후 **소셜 계정을 추가로 연동**할 수 있도록 구성되어 있습니다.

`추후에 전체적으로 디자인 및 기능이 변경될 예정. 참고만 하길 바람`

---

## 로그인 방식

### 1. 일반 로그인

- 학생들의 **그린 마일리지 계정**을 사용합니다.
- 로그인 시 필요한 정보는 **아이디와 비밀번호**입니다.
- **그린 마일리지 계정**은 이 프로젝트에서 직접 관리하지 않습니다.

### 2. 소셜 로그인

- 일반 로그인 완료 후, **소셜 계정 인증을 통해 로그인 연동**이 가능합니다.
- 지원하는 소셜 로그인 플랫폼은 다음과 같습니다.
  - GitHub

---

## 기타 사항

- 아이디 및 비밀번호 **변경 기능은 제공하지 않습니다.**
- 처음 로그인한 사용자에게는 **기본 프로필 이미지로 [identicon](https://www.dicebear.com/styles/identicon/) 이미지가 자동 설정**됩니다.
