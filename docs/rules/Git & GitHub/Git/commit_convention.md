---
title: 커밋 컨벤션
description: 커밋에 관련된 컨벤션입니다.
---

> 특정 파일 혹은 폴더를 깃에 커밋할 경우, 반드시 아래 컨벤션을 참고하여 커밋을 작성해주시길 바랍니다.  
> 예시 : `feat: 로그인 기능 추가`

| **종류** | **설명**                                          |
| -------- | ------------------------------------------------- |
| feat     | 새로운 기능 추가                                  |
| fix      | 버그 수정                                         |
| docs     | 문서 수정                                         |
| style    | 코드 스타일 변경 ( 코드 포매팅, 세미콜론 누락 등) |
| design   | 사용자 UI 디자인 변경 (CSS 등)                    |
| test     | 테스트 코드, 리팩토링 (Test Code)                 |
| refactor | 리팩토링                                          |
| build    | 빌드 관련 파일 추가 및 수정                       |
| ci       | CI 설정 관련 파일 추가 및 수정                    |
| perf     | 성능 개선                                         |
| chore    | 자잘한 수정이나 빌드 업데이트                     |
| rename   | 파일명 혹은 폴더명만 수정한 경우                  |
| remove   | 파일 삭제                                         |
