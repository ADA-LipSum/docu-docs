`.prettierrc.js` 또는 `.prettierrc.ts`는 다음과 같이 통일합니다.

```
module.exports = {
  semi: true,
  printWidth: 120,
  endOfLine: 'auto',
  singleQuote: true,
  useTabs: false,
  tabWidth: 2,
  trailingComma: 'all',
  arrowParens: 'always'
};
```
